# 💰 Personal Finance Tracker

A CLI application to track personal income and expenses built with Python OOP.

## Features
- Add income and expenses
- View balance and statistics
- Filter by category
- Monthly report
- Save and load data (JSON)

## Project Structure
finance_tracker/
├── main.py
├── models/
│   ├── transaction.py
│   └── wallet.py
└── utils/
    ├── file_manager.py
    └── cli.py

## Technologies
- Python 3
- OOP (Classes, Inheritance)
- JSON file handling
